# 🔈 Bubble-Hit-Game

A simple game application where user can play **Bubble-Hit**.

## 🚀 Features

- ✅ Hit the bubble
- ✅ Timer
- ✅ Hit Score
- ✅ Displays:
  - Bubbles inside container
- ✅ Simple and responsive design using HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

- **HTML** — Page structure
- **CSS** — Styling and layout
- **JavaScript** — dynamic DOM updates
